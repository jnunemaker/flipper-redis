# Flipper Redis

A redis adapter for [Flipper](https://github.com/jnunemaker/flipper), the feature flipping gems.

## Installation

Add this line to your application's Gemfile:

    gem 'flipper-redis'

And then execute:

    $ bundle

Or install it yourself with:

    $ gem install flipper-redis
